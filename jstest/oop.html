变量范围

function f(){local=2}
该变量没有使用var声明，在没有调用该f()函数前，该变量不存在。
当调用该f()函数后，该变量的范围是全局的。

函数是数据

function f(){return 1;}
var f=function(){return 1;}

1、Using anonymous functions(匿名函数)

function(a){return 1;}

2、Callbacks functions(回调函数)

function one(){return 1;}
function two(){return 2;}
function three(a,b){
return a()+b();
}
(当传递函数A到另一个函数B中，由B执行A，那么就说A是回调函数，
如果A是匿名，那么就是匿名回调函数)

3、Self-invoking functions(自我调用函数)

(function(){
alert('boo');
})()
//该匿名函数，在定义完后就自我调用

4、Inner functions (functions defined inside functions)(内部函数)

function a(param){
function b(input){
return input*2;
}
return 'the result is '+b(param);
}
或者
function a(param){
var b=function(input){
return input*2;
}
return 'the result is '+b(param);
}

5、Functions that return functions (返回一个函数的函数)

function a(){
alert('A!');
return function(){
alert('B!');
}
}
var newFunc=a();//A!
newFunc();//B!

6、Functions that redefine themselves(重新定义自己函数)

function a(){
alert('A!');
a=function(){
alert('B!');
}
}

7、Closures(闭包函数)

function f(){
var b="b";//局部变量
return function(){
return b;
}
}
var n=f();
n();//b
(闭包:在全局范围内访问函数的私有变量)
